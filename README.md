# Furniture Store
Full-stack furniture e-commerce app built with React, TypeScript, Node.js, PostgreSQL, and Prisma.

## Опис проекту
**Furniture Store** — це сучасний Full-Stack інтернет-магазин меблів для портфоліо.  
Користувачі можуть переглядати товари, додавати їх у кошик та оформлювати замовлення.  
Адміністратор керує товарами та замовленнями.  

---

## Функціональні можливості (MVP)
- Каталог товарів з фільтрами та пошуком
- Сторінка товару з деталями
- Кошик користувача
- Авторизація/Реєстрація
- Оформлення замовлення
- Адмін-панель (планується)

---

## Технології
### Frontend:
- React + TypeScript
- Tailwind CSS
- React Router

### Backend:
- Node.js + Express + TypeScript
- JWT для авторизації
- PostgreSQL + Prisma ORM


### Інструменти:
- Git + GitHub
- VS Code
- Postman
- Vercel / Render
