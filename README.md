# Furniture Store
Full-stack furniture e-commerce app built with React, TypeScript, Node.js, PostgreSQL, and Prisma.

## Project description
**Furniture Store** is a modern full-stack online furniture store for portfolios.  
Users can browse products, add them to their cart, and place orders.  
The administrator manages products and orders.  

---

## Features (MVP)
- Product catalog with filters and search
- Product page with details
- User shopping cart
- Authorization/Registration
- Order placement
- Admin panel (planned)

---

## Technologies
### Frontend:
- React + TypeScript
- Tailwind CSS
- React Router

### Backend:
- Node.js + Express + TypeScript
- JWT for authorization
- PostgreSQL + Prisma ORM
